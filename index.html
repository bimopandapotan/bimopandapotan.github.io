Latest Projects

Student Registration

Explore PHP in greater depth.

This project is a web-based student registration system built with PHP. It allows students to input their personal information, parents' details, and academic scores as part of the school admission process. The system also includes an automatic selection feature that filters applicants based on predefined score criteria, making the registration and evaluation process more efficient and accurate.

AnimeSea

Android app group project.

AnimeSea is a group-developed Android application built using Java. The app allows users to create and manage accounts, search for anime titles, view detailed descriptions fetched from an external API, and save their favorite shows using the bookmark feature. With a clean user interface and responsive search functionality, AnimeSea provides anime enthusiasts with a convenient platform to explore and organize their favorite series on mobile.

Open Weather

Gain deeper insight into working with APIs.

Open Weather is a simple Python-based application that allows users to retrieve real-time weather information by entering a city name. The app utilizes the OpenWeather API to fetch data such as temperature, weather conditions, and humidity. This project is ideal for those learning how to work with external APIs in Python, including handling HTTP requests, parsing JSON responses, and displaying data in a user-friendly format.

NotNotes

Explore one of the many powerful web frameworks.

NotNotes is a web application built using the Laravel framework, designed to help users manage their personal notes efficiently. The app includes core features such as adding new notes, editing existing ones, and deleting notes when no longer needed. This project demonstrates the power of Laravel in building dynamic, CRUD-based web applications while promoting clean architecture, routing, and database interaction.
